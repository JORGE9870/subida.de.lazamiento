# 🚀 Instrucciones Rápidas - VIP Scorts Countdown

## ⚡ Pasos para ejecutar el proyecto:

### 1. Instalar dependencias
```bash
npm install
```

### 2. Ejecutar en modo desarrollo
```bash
npm run serve
```

### 3. Abrir en el navegador
```
http://localhost:8080
```
